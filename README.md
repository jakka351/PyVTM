# PyVTM
A tribute to 'Vehicle Traffic Monitor" by Ford Motor Company
  

A recreation of the application by Ford Motor Company VTM, Vehicle Traffic Monitor, using Python and TKinter with a 32 Bit Python J2534 Implementation.
Staying true to the original, the password for the Read/Write section is `cancan`. Enjoy.
